# Object Detection + Distance Estimation for Robotics Navigation
Assignment Submission – Khushi Saraswat

## Project Overview
This project detects navigation objects such as cones, barriers,
and stop signs and estimates their distance from the robot camera.

## Model
YOLOv8 nano model was used for object detection.

## Distance Estimation
Distance is estimated using camera geometry.

Distance = (Real Height × Focal Length) / Pixel Height

## Output
Objects are detected with bounding boxes and annotated
with estimated distance.
